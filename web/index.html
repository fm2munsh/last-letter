How to play

Overview

Last letter is a 2-player word game that runs through the kik messenger. The game requires one person to pick a category and start off by sending a word from that category to his opponent. The opponent then has to respond with a word from the same category that starts with the last letter from the previous word.

Rules

Each player is given 3 "lives", a player loses a life when he enters a word that is not part of the category, does not start with last letter of the previous word or has already been used in the current game. The game ends when a player loses all 3 of his lives.

Categories

Currently the categories supported include: Countries and the Dictionary.
